# Rtvgb – Networking Labs

Hands-on networking labs built in Cisco Packet Tracer and other tools, focused on building skills toward cloud security and network engineering roles.

## Labs

| Lab | Topic | Status |
|-----|-------|--------|
| [Lab 1 – Structured Cabling](./Lab1_Structured_Cabling/) | Designed and implemented a structured cabling system for a 7-office corporate environment in Cisco Packet Tracer | ✅ Complete |
| [Lab 2 – DHCP, DNS & Single Switch Network](./Lab2_DHCP_DNS_Single_Switch/) | Configured a single-switch network with DHCP, DNS, and web servers using 192.168.200.0/24 | ✅ Complete |
| [Lab 3 – Default Gateway Configuration](./Lab3_Default_Gateway_Configuration/) | Configured default gateways on all end devices to enable inter-network communication | ✅ Complete |

## Environment
- **Simulation Tool:** Cisco Packet Tracer
- **Focus Areas:** Physical infrastructure, IP addressing, switching, DHCP, DNS, routing, default gateways
